Step 2 — Set Up .gitignore for Unity (Critical)
Unity generates massive auto-generated files that should NOT be pushed to GitHub.
In your repository click Add file → Create new file
Name it exactly .gitignore
Paste this content:

[Ll]ogs/
[Mm]emoryCaptures/
